How to build:
- Once you've installed tsc, literally just run "tsc" in terminal in the gmtk-gamejam-2023 folder

How to test locally:
- In the gmtk-gamejam-2023 folder, run "python -m localserver.py"
- Go to http://localhost:8000/index.html in your browser
- Changes are reflected once you rebuild via tsc and, I believe, reload the page
